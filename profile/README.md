# Atlas Builder 🟧

Atlas Builder is a research and development organization building the first MEV block builder infrastructure for PulseChain. Our mission is to eliminate toxic MEV on PulseChain, maximize validator rewards, and return value to the users and ecosystem that generate it.

MEV bots have drained millions of dollars from PulseChain users since day one with zero protection in place. Atlas Builder changes that.


## Our Approach

On Ethereum, MEV-Boost increased staking rewards by 42% and pays validators 3x more per block on average. 90% of all Ethereum validators run it. Atlas Builder brings this same proven infrastructure to PulseChain for the first time.

We build in public and believe PulseChain deserves real infrastructure.

## Infrastructure

- **MEV-Boost** — Open source sidecar allowing validators to outsource block building and propose more profitable blocks. Live in beta now.
- **MEV-Boost Relay** — Open source relay connecting validators to the Atlas block builder.
